## ğŸ”¥ Firebase Hosting Setup

1. **Login to Firebase CLI:**
   node_modules/.bin/firebase login

2. **Initialize Firebase Hosting:**
   node_modules/.bin/firebase init

   node_modules/.bin/firebase deploy
